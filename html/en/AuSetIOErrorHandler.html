Name
       AuSetIOErrorHandler - set handler for fatal IO errors

Synopsis
       #include <audio/audiolib.h>

       AuBool (*AuSetIOErrorHandler(server, handler))()
           AuServer *server;
           AuBool (*handler)(AuServer *);

Arguments
       server    Specifies the connection to the audio server.

       handler   Specifies  the  fatal error handler.  Use NULL to reinstall the
                 default fatal error handler.

Description
       AuSetIOErrorHandler sets the fatal I/O error handler and returns the pre‐
       vious  error handler.  The error handler will be invoked if a system call
       error occurs (for example, the connection to the server  was  lost).   As
       this is assumed to be a fatal condition, the error handler routine should
       not return.  If the I/O error handler does  return,  the  client  process
       exits.

See Also

       AuSynchronize.
