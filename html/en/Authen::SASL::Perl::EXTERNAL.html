NAME
       Authen::SASL::Perl::EXTERNAL - External Authentication class

SYNOPSIS
         use Authen::SASL qw(Perl);

         $sasl = Authen::SASL->new(
           mechanism => 'EXTERNAL',
           callback  => {
             user => $user
           },
         );

DESCRIPTION
       This method implements the client part of the EXTERNAL SASL algorithm, as
       described in RFC 2222.

   CALLBACK
       The callbacks used are:

       user
           The username to be used for authentication
