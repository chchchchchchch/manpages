NAME
       DateTime::Locale::he_IL - DateTime locale package for "he_IL"

SYNOPSIS
         use DateTime;

         my $dt = DateTime->now( locale => 'he_IL' );
         print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Hebrew Israel.
