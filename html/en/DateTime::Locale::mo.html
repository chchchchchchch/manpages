NAME
       DateTime::Locale::mo - DateTime locale package for "mo"

SYNOPSIS
         use DateTime;

         my $dt = DateTime->now( locale => 'mo' );
         print $dt->month_name();

DESCRIPTION
       This is the DateTime locale package for Moldavian.

DATA
       This locale inherits from the DateTime::Locale::ro_MD locale.

       It contains the following data.
