NAME
       FcAtomicDeleteNew - delete new file

SYNOPSIS
       #include <fontconfig.h>

       void FcAtomicDeleteNew (FcAtomic *atomic);

DESCRIPTION
       Deletes the new file.  Used in error recovery to back out changes.

VERSION
       Fontconfig version 2.9.0
