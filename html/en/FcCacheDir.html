NAME
       FcCacheDir - Return directory of cache

SYNOPSIS
       #include <fontconfig.h>

       const FcChar8 * FcCacheDir (const FcCache *cache);

DESCRIPTION
       This function returns the directory from which the cache was constructed.

VERSION
       Fontconfig version 2.9.0
