NAME
       FcCharSetCreate - Create an empty character set

SYNOPSIS
       #include <fontconfig.h>

       FcCharSet * FcCharSetCreate (void);

DESCRIPTION
       FcCharSetCreate  allocates  and  initializes  a  new  empty character set
       object.

VERSION
       Fontconfig version 2.9.0
