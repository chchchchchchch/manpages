NAME
       FcDirCacheValid - check directory cache

SYNOPSIS
       #include <fontconfig.h>

       FcBool FcDirCacheValid (const FcChar8 *dir);

DESCRIPTION
       Returns  FcTrue  if  dir has an associated valid cache file, else returns
       FcFalse

VERSION
       Fontconfig version 2.9.0
