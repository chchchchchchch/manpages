NAME
       FcFontSetDestroy - Destroy a font set

SYNOPSIS
       #include <fontconfig.h>

       void FcFontSetDestroy (FcFontSet *s);

DESCRIPTION
       Destroys  a font set.  Note that this destroys any referenced patterns as
       well.

VERSION
       Fontconfig version 2.9.0
