NAME
       FcInitLoadConfig - load configuration

SYNOPSIS
       #include <fontconfig.h>

       FcConfig * FcInitLoadConfig (void);

DESCRIPTION
       Loads the default configuration file and returns the resulting configura‐
       tion.  Does not load any font information.

VERSION
       Fontconfig version 2.9.0
