NAME
       FcLangSetDestroy - destroy a langset object

SYNOPSIS
       #include <fontconfig.h>

       void FcLangSetDestroy (FcLangSet *ls);

DESCRIPTION
       FcLangSetDestroy  destroys a FcLangSet object, freeing all memory associ‐
       ated with it.

VERSION
       Fontconfig version 2.9.0
