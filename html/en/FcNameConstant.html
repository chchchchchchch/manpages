NAME
       FcNameConstant - Get the value for a symbolic constant

SYNOPSIS
       #include <fontconfig.h>

       FcBool FcNameConstant (FcChar8 *string, int *result);

DESCRIPTION
       Returns whether a symbolic constant with name string is registered, plac‐
       ing the value of the constant in result if present.

VERSION
       Fontconfig version 2.9.0
