NAME
       FcPatternCreate - Create a pattern

SYNOPSIS
       #include <fontconfig.h>

       FcPattern * FcPatternCreate (void);

DESCRIPTION
       Creates  a  pattern  with  no  properties;  used  to  build patterns from
       scratch.

VERSION
       Fontconfig version 2.9.0
