NAME
       FcPatternEqual - Compare patterns

SYNOPSIS
       #include <fontconfig.h>

       FcBool FcPatternEqual (const FcPattern *pa, const FcPattern *pb);

DESCRIPTION
       Returns whether pa and pb are exactly alike.
