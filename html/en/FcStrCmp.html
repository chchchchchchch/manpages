NAME
       FcStrCmp - compare UTF-8 strings

SYNOPSIS
       #include <fontconfig.h>

       int FcStrCmp (const FcChar8 *s1, const FcChar8 *s2);

DESCRIPTION
       Returns the usual <0, 0, >0 result of comparing s1 and s2.

VERSION
       Fontconfig version 2.9.0
