NAME
       FcStrFree - free a string

SYNOPSIS
       #include <fontconfig.h>

       void FcStrFree (FcChar8 *s);

DESCRIPTION
       This  is  just a wrapper around free(3) which helps track memory usage of
       strings within the fontconfig library.

VERSION
       Fontconfig version 2.9.0
