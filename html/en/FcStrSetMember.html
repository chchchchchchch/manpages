NAME
       FcStrSetMember - check set for membership

SYNOPSIS
       #include <fontconfig.h>

       FcBool FcStrSetMember (FcStrSet *set, const FcChar8 *s);

DESCRIPTION
       Returns whether s is a member of set.

VERSION
       Fontconfig version 2.9.0
