NAME
       FcToLower - convert upper case ASCII to lower case

SYNOPSIS
       #include <fontconfig.h>

       FcChar8 FcToLower (FcChar8c);

DESCRIPTION
       This  macro converts upper case ASCII c to the equivalent lower case let‐
       ter.

VERSION
       Fontconfig version 2.9.0
