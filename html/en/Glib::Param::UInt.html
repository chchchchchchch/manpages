NAME
       Glib::Param::UInt -  Wrapper for uint parameters in GLib

HIERARCHY
         Glib::ParamSpec
         +----Glib::Param::UInt

         Glib::ParamSpec
         +----Glib::Param::UChar

         Glib::ParamSpec
         +----Glib::Param::ULong

DESCRIPTION
       This page documents the extra accessors available for all of the unsigned
       integer type paramspecs: UChar, UInt, and ULong.  Perl really only
       supports full-size integers, so all of these methods return UVs; the
       distinction of integer size is important to the underlying C library and
       also determines the data value range.

METHODS
   unsigned = $pspec->get_maximum
   unsigned = $pspec->get_minimum
