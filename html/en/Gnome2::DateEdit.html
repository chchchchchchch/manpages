       'dateedit-flags' (Gnome2::DateEditFlags : default show-time : readable /
       writable)
           Flags for how DateEdit looks

       'initial-time' (Glib::ULong : default 0 : readable / writable)
           The initial time

       'lower-hour' (integer : default 7 : readable / writable)
           Lower hour in the time popup selector

       'time' (Glib::ULong : default 0 : readable / writable)
           The time currently selected

       'upper-hour' (integer : default 19 : readable / writable)
           Upper hour in the time popup selector
