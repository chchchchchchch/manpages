PROPERTIES
       'font' (gpointer : default 0 : readable)
           The selected GtkFont

       'font-name' (string : default "sans 12" : readable / writable)
           Name of the selected font

       'label-font-size' (integer : default 14 : readable / writable)
           Font size for label in font info mode

       'mode' (Gnome2::FontPickerMode : default "pixmap" : readable / writable)
           The mode of operation of the font picker

       'preview-text' (string : default
       "AaBbCcDdEeFfGgHhIiJjKkLlMmNnOoPpQqRrSsTtUuVvWwXxYyZz" : readable /
       writable)
           Preview text shown in the dialog

       'show-size' (boolean : default true : readable / writable)
           Show size in font info mode

       'title' (string : default "Pick a Font" : readable / writable)
           The title of the selection dialog box

       'use-font-in-label' (boolean : default false : readable / writable)
           Use font in the label in font info mode
