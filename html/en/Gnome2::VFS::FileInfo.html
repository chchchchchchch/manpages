NAME
       Gnome2::VFS::FileInfo - stores information about files, like stat

METHODS
   fileinfo = Gnome2::VFS::FileInfo->new ($hash_ref)
       ·   $hash_ref (scalar)

       Creates a new GnomeVFSFileInfo object from hash_ref for use with
       Gnome2::VFS::FileInfo::matches, for example.  Normally, you can always
       directly use a hash reference if you're asked for a GnomeVFSFileInfo.

   boolean = $a->matches ($b)
       ·   $b (Gnome2::VFS::FileInfo)
