NAME
       Log::Log4perl::InternalDebug - Dark Magic to enable _INTERNAL_DEBUG

DESCRIPTION
       When called with

           perl -MLog::Log4perl::InternalDebug t/001Test.t

       scripts will run with _INTERNAL_DEBUG set to a true value and hence print
       internal Log4perl debugging information.
