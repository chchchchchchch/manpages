NAME
       Net::DNS::RR::A - DNS A resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Address (A) resource records.

METHODS
   address
           print "address = ", $rr->address, "\n";

       Returns the RR's address field.
