NAME
       Net::DNS::RR::CNAME - DNS CNAME resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Canonical Name (CNAME) resource records.

METHODS
   cname
           print "cname = ", $rr->cname, "\n";

       Returns the RR's canonical name.
