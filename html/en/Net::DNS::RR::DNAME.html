NAME
       Net::DNS::RR::DNAME - DNS DNAME resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Non-Terminal Name Redirection (DNAME) resource records.

METHODS
   dname
           print "dname = ", $rr->dname, "\n";

       Returns the DNAME target.
