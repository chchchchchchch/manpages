NAME
       Net::DNS::RR::HINFO - DNS HINFO resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Host Information (HINFO) resource records.

METHODS
   cpu
           print "cpu = ", $rr->cpu, "\n";

       Returns the CPU type for this RR.

   os
           print "os = ", $rr->os, "\n";

       Returns the operating system type for this RR.
