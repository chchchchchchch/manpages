NAME
       Net::DNS::RR::ISDN - DNS ISDN resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS ISDN resource records.

METHODS
   address
           print "address = ", $rr->address, "\n";

       Returns the RR's address field.

   sa
           print "subaddress = ", $rr->sa, "\n";

       Returns the RR's subaddress field.
