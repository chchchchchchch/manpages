NAME
       Net::DNS::RR::MR - DNS MR resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       Class for DNS Mail Rename (MR) resource records.

METHODS
   newname
           print "newname = ", $rr->newname, "\n";

       Returns the RR's new name field.
