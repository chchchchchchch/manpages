NAME
       Net::DNS::RR::SPF - DNS SPF resource record

SYNOPSIS
       "use Net::DNS::RR";

DESCRIPTION
       This is a clone of the TXT record. This class therfore completely
       inherits all properties of the Net::DNS::RR::TXT class.

       Please see the Net::DNS::RR::TXT perldocumentation for details
