NAME
       Net::DNS::Resolver::Win32 - Windows Resolver Class

SYNOPSIS
        use Net::DNS::Resolver;

DESCRIPTION
       This class implements the windows specific portions of
       "Net::DNS::Resolver".

       No user serviceable parts inside, see Net::DNS::Resolver for all your
       resolving needs.
