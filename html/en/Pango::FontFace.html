NAME
       Pango::FontFace - object for Pango representing a group of fonts varying
       only in size

HIERARCHY
         Glib::Object
         +----Pango::FontFace

METHODS
   fontdescription = $face->describe
   string = $face->get_face_name
   boolean = $face->is_synthesized
       Since: pango 1.18

   @sizes = $face->list_sizes
       List the sizes available for a bitmapped font.  For scalable fonts, this
       will return an empty list.

       Since: pango 1.4
