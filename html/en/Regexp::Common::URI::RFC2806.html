NAME
       Regexp::Common::URI::RFC2806 -- Definitions from RFC2806;

SYNOPSIS
           use Regexp::Common::URI::RFC2806 qw /:ALL/;

DESCRIPTION
       This package exports definitions from RFC2806. It's intended usage is for
       Regexp::Common::URI submodules only. Its interface might change without
       notice.

REFERENCES
       [RFC 2616]
           Fielding, R., Gettys, J., Mogul, J., Frystyk, H., Masinter, L.,
           Leach, P. and Berners-Lee, Tim: Hypertext Transfer Protocol --
           HTTP/1.1.  June 1999.
