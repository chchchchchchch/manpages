NAME
       SDL_EventState  - This function allows you to set the state of processing
       certain events.

SYNOPSIS
       #include "SDL.h"

       Uint8 SDL_EventState(Uint8 type, int state);

DESCRIPTION
       This function allows you to set the state  of  processing  certain  event
       type's.

       If  state  is  set  to  SDL_IGNORE, that event type will be automatically
       dropped from the event queue and will not be filtered.

       If state is set to SDL_ENABLE, that event type  will  be  processed  nor‐
       mally.

       If state is set to SDL_QUERY, SDL_EventState will return the current pro‐
       cessing state of the specified event type.

       A list of event type's can be found in the SDL_Event section.
