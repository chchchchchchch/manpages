NAME
       SDL_FreeCursor - Frees a cursor created with SDL_CreateCursor.

SYNOPSIS
       #include "SDL.h"

       void SDL_FreeCursor(SDL_Cursor *cursor);

DESCRIPTION
       Frees a SDL_Cursor that was created using SDL_CreateCursor.
