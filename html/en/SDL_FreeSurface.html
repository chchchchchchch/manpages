NAME
       SDL_FreeSurface - Frees (deletes) a SDL_Surface

SYNOPSIS
       #include "SDL.h"

       void SDL_FreeSurface(SDL_Surface *surface);

DESCRIPTION
       Frees the resources used by a previously created SDL_Surface. If the sur‐
       face was created using SDL_CreateRGBSurfaceFrom then the  pixel  data  is
       not freed.
