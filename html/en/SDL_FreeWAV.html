NAME
       SDL_FreeWAV - Frees previously opened WAV data

SYNOPSIS
       #include "SDL.h"

       void SDL_FreeWAV(Uint8 *audio_buf);

DESCRIPTION
       After  a  WAVE file has been opened with SDL_LoadWAV its data can eventu‐
       ally be freed with SDL_FreeWAV. audio_buf is a pointer to the buffer cre‐
       ated by SDL_LoadWAV.
