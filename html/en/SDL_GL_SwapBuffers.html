NAME
       SDL_GL_SwapBuffers - Swap OpenGL framebuffers/Update Display

SYNOPSIS
       #include "SDL.h"

       void SDL_GL_SwapBuffers(void );

DESCRIPTION
       Swap the OpenGL buffers, if double-buffering is supported.
