NAME
       SDL_GetKeyName - Get the name of an SDL virtual keysym

SYNOPSIS
       #include "SDL.h"

       char *SDL_GetKeyName(SDLKey key);

DESCRIPTION
       Returns the SDL-defined name of the SDLKey key.
