NAME
       SDL_GetThreadID - Get the SDL thread ID of a SDL_Thread

SYNOPSIS
       #include "SDL.h" #include "SDL_thread.h"

       Uint32 SDL_GetThreadID(SDL_Thread *thread);

DESCRIPTION
       Returns the ID of a SDL_Thread created by SDL_CreateThread.
