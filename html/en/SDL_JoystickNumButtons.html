NAME
       SDL_JoystickNumButtons - Get the number of joysitck buttons

SYNOPSIS
       #include "SDL.h"

       int SDL_JoystickNumButtons(SDL_Joystick *joystick);

DESCRIPTION
       Return  the number of buttons available from a previously opened SDL_Joy‐
       stick.

RETURN VALUE
       Number of buttons.

SEE ALSO
       SDL_JoystickGetButton, SDL_JoystickOpen
