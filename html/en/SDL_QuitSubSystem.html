NAME
       SDL_QuitSubSystem - Shut down a subsystem

SYNOPSIS
       #include "SDL.h"

       void SDL_QuitSubSystem(Uint32 flags);

DESCRIPTION
       SDL_QuitSubSystem  allows you to shut down a subsystem that has been pre‐
       viously initialized by SDL_Init or  SDL_InitSubSystem.  The  flags  tells
       SDL_QuitSubSystem  which subsystems to shut down, it uses the same values
       that are passed to SDL_Init.
