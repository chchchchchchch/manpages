NAME
       SDL_WM_SetCaption - Sets the window tile and icon name.

SYNOPSIS
       #include "SDL.h"

       void SDL_WM_SetCaption(const char *title, const char *icon);

DESCRIPTION
       Sets the title-bar and icon name of the display window.
