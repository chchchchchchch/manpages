NAME
       SDL_WarpMouse - Set the position of the mouse cursor.

SYNOPSIS
       #include "SDL.h"

       void SDL_WarpMouse(Uint16 x, Uint16 y);

DESCRIPTION
       Set the position of the mouse cursor (generates a mouse motion event).
