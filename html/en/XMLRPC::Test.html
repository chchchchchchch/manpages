NAME
       XMLRPC::Test - Test framework for XMLRPC::Lite

SYNOPSIS
         use XMLRPC::Test;

         XMLRPC::Test::Server::run_for('http://localhost/cgi-bin/XMLRPC.cgi');

DESCRIPTION
       XMLRPC::Test provides simple framework for testing server
       implementations.  Specify your address (endpoint) and run provided tests
       against your server.  See t/1*.t for examples.
