NAME
       XPutBackEvent - put events back on the queue

SYNTAX
       XPutBackEvent(Display *display, XEvent *event);

ARGUMENTS
       display   Specifies the connection to the X server.

       event     Specifies the event.

DESCRIPTION
       The XPutBackEvent function pushes an event back onto the head of the dis‐
       play's event queue by copying the event into the queue.  This can be use‐
       ful if you read an event and then decide that you would rather deal with
       it later.  There is no limit to the number of times in succession that
       you can call XPutBackEvent.
