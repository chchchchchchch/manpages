NAME
       XkbGetXlibControls - Determines the current state of the Library Controls

SYNOPSIS
       unsigned int XkbGetXlibControls (Display  *display);

ARGUMENTS
       - display
              connection to X server

DESCRIPTION
       XkbGetXlibControls returns the current state of the Library Controls as a
       bit mask that is an inclusive OR of the control masks from  Table  1  for
       the  controls  that are enabled. For the optional compose processing con‐
       trols, the fact that a control is enabled does not imply that it is actu‐
       ally implemented.

       NewKeyboards (1  <<  3)  XkbLC_ConsumeKeysOnComposeFail     (1   <<   29)
