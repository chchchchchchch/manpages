NAME
       XtToolkitThreadInitialize - initialize the toolkit for multiple threads

SYNTAX
       Boolean XtToolkitThreadInitialize(void);

DESCRIPTION
       If XtToolkitThreadInitialize was previously called, it returns. The
       application programmer must ensure that two or more threads do not simul‐
       taneously attempt to call XtToolkitThreadInitialize; the effect of this
       is undefined.  XtToolkitThreadInitialize returns True if the host operat‐
       ing system has threads and the Intrinsics are thread safe.
