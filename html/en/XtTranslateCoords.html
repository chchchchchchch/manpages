NAME
       XtTranslateCoords - translate widget coordinates

SYNTAX
       void XtTranslateCoords(Widget w, Position x, y, Position *rootx_return,
              *rooty_return);

ARGUMENTS
       rootx_return
       rooty_return
                 Returns the root-relative x and y coordinates.

       x
       y         Specify the widget-relative x and y coordinates.

       w         Specifies the widget.

DESCRIPTION
       While XtTranslateCoords is similar to the Xlib XTranslateCoordinates
       function, it does not generate a server request because all the required
       information already is in the widget's data structures.
