NAME
       aa_attrs - returns pointer to the text output buffer used by AA-lib.

SYNOPSIS
       #include <aalib.h>

       char *aa_attrs(aa_context *a);

PARAMETERS
       aa_context *a
              Specifies the AA-lib context to operate on.

DESCRIPTION
       The  attribute output buffer is simple array of characters specifying the
       attributes.  The array is organizated in the  aa_scrheight  (a)  rows  of
       aa_scrwidth(a) characters.

RETURNS
       Pointer the text output buffer used by AA-lib.
