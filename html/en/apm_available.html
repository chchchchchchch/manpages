NAME
       apm_available - test whether APM subsystem is available


SYNOPSIS
       apm_available


DESCRIPTION
       apm_available  checks  whether the APM ("Advanced Power Management") sub‐
       system is available, as evidenced by the presence of the /proc/apm file.


OPTIONS
       None.

EXIT STATUS
       0 (true)       APM subsystem is available
       1 (false)      APM subsystem is not available
       2              usage error (arguments supplied)

FILES
       /proc/apm         APM status information file
