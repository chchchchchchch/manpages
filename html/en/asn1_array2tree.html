NAME
       asn1_array2tree - API function

SYNOPSIS
       #include <libtasn1.h>

       asn1_retCode  asn1_array2tree(const  ASN1_ARRAY_TYPE * array, ASN1_TYPE *
       definitions, char * errorDescription);

ARGUMENTS
       const ASN1_ARRAY_TYPE * array
                   specify the array that contains ASN.1 declarations

       ASN1_TYPE * definitions
                   return the pointer to the structure created by  *ARRAY  ASN.1
                   declarations

       char * errorDescription
                   return the error description.

DESCRIPTION
       Creates  the structures needed to manage the ASN.1 definitions.  array is
       a vector created by asn1_parser2array().

RETURNS
       ASN1_SUCCESS if structure was created  correctly,  ASN1_ELEMENT_NOT_EMPTY
       if  *definitions not ASN1_TYPE_EMPTY, ASN1_IDENTIFIER_NOT_FOUND if in the
       file there is an identifier that is not defined (see errorDescription for
       more  information),  ASN1_ARRAY_ERROR  if  the  array pointed by array is
       wrong.
