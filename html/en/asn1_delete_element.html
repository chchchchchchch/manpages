NAME
       asn1_delete_element - API function

SYNOPSIS
       #include <libtasn1.h>

       asn1_retCode  asn1_delete_element(ASN1_TYPE  structure, const char * ele‐
       ment_name);

ARGUMENTS
       ASN1_TYPE structure
                   pointer to the structure that contains the element  you  want
                   to delete.

       const char * element_name
                   element's name you want to delete.

DESCRIPTION
       Deletes the element named *element_name inside *structure.

RETURNS
       ASN1_SUCCESS  if  successful,  ASN1_ELEMENT_NOT_FOUND if the element_name
       was not found.
