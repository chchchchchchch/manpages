NAME
       asn1_get_tag_der - API function

SYNOPSIS
       #include <libtasn1.h>

       int  asn1_get_tag_der(const  unsigned  char  * der, int der_len, unsigned
       char * cls, int * len, unsigned long * tag);

ARGUMENTS
       const unsigned char * der
                   DER data to decode.

       int der_len Length of DER data to decode.

       unsigned char * cls
                   Output variable containing decoded class.

       int * len   Output variable containing the length of the DER TAG data.

       unsigned long * tag
                   Output variable containing the decoded tag.

DESCRIPTION
       Decode the class and TAG from DER code.

RETURNS
       Returns ASN1_SUCCESS on success, or an error.
