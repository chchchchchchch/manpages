NAME
       asn1_number_of_elements - API function

SYNOPSIS
       #include <libtasn1.h>

       asn1_retCode  asn1_number_of_elements(ASN1_TYPE  element,  const  char  *
       name, int * num);

ARGUMENTS
       ASN1_TYPE element
                   pointer to the root of an ASN1 structure.

       const char * name
                   the name of a sub-structure of ROOT.

       int * num   pointer to an integer where the result will be stored

DESCRIPTION
       Counts the number of elements of a sub-structure called NAME  with  names
       equal to "?1","?2", ...

RETURNS
       ASN1_SUCCESS  if successful, ASN1_ELEMENT_NOT_FOUND if name is not known,
       ASN1_GENERIC_ERROR if pointer num is NULL.
