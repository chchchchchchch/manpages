NAME
       asn1_perror - API function

SYNOPSIS
       #include <libtasn1.h>

       void asn1_perror(asn1_retCode error);

ARGUMENTS
       asn1_retCode error
                   is an error returned by a libtasn1 function.

DESCRIPTION
       Prints  a string to stderr with a description of an error.  This function
       is like perror().  The only  difference  is  that  it  accepts  an  error
       returned by a libtasn1 function.

       This function replaces libtasn1_perror() in older libtasn1.

       1.6
