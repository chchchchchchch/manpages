NAME
       yacc - GNU Project parser generator

SYNOPSIS
       yacc [OPTION]... FILE

DESCRIPTION
       Yacc (Yet Another Compiler Compiler) is a parser generator.  This version
       is a simple wrapper around bison(1).  It  passes  option  -y,  --yacc  to
       activate  the  upward compatibility mode.  See bison(1) for more informa‐
       tion.
