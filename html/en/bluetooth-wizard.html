NAME
       bluetooth-wizard - GTK wizard for setting up devices with the Linux Blue‐
       tooth stack

SYNOPSIS
       bluetooth-wizard

DESCRIPTION
       bluetooth-wizard will display a wizard for setting up Bluetooth  devices.
       bluetooth-wizard is part of bluez-gnome, see also http://www.bluez.org

OPTIONS
       bluetooth-wizard takes no options
