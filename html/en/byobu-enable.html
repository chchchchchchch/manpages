NAME
       byobu-enable, byobu-disable - wrapper script for enabling/disabling auto‐
       matic startup of byobu after login into text console


SYNOPSIS
       byobu-enable

       byobu-disable


DESCRIPTION
       byobu-enable will do two things:
        1) run byobu-launcher-install(1) to configure byobu(1) to  launch  auto‐
       matically on login
        2) then launch byobu(1)

       byobu-disable will simply run byobu-launcher-uninstall(1).
