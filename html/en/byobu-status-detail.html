NAME
       byobu-status-detail - Wrapper that uses a sensible pager


DESCRIPTION
       byobu-status-detail  is  a  simple  script that uses a sensible pager for
       displaying the detailed status of all byobu status scripts.


       http://launchpad.net/byobu
