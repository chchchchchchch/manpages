NAME
        calibre - launch Calibre Graphical User Interface

USAGE
       calibre [opts] [path_to_ebook]

DESCRIPTION
       Launch  the  main calibre Graphical User Interface and optionally add the
       ebook at path_to_ebook to the database.

       Whenever you pass arguments to calibre that have spaces in them,  enclose
       the arguments in quotation marks.

MAIN OPTIONS
       --version
              show program´s version number and exit

       -h, --help
              show this help message and exit

       --redirect-console-output
              Redirect  console  output  to  a  dialog  window  (both stdout and
              stderr). Useful on windows where GUI apps do  not  have  a  output
              streams.

       --with-library=WITH_LIBRARY
              Use the library located at the specified path.

       -v, --verbose
              Log debugging information to console
