NAME
       cherryd - cherryd

SYNOPSIS
       cherryd [options]

OPTIONS
       -h, --help
              show this help message and exit

       -c CONFIG, --config=CONFIG
              specify config file(s)

       -d     run the server as a daemon

       -e ENVIRONMENT, --environment=ENVIRONMENT
              apply the given config environment

       -f     start a fastcgi server instead of the default HTTP server

       -s     start a scgi server instead of the default HTTP server

       -x     start a cgi server instead of the default HTTP server

       -i IMPORTS, --import=IMPORTS
              specify modules to import

       -p PIDFILE, --pidfile=PIDFILE
              store the process id in the given file

       -P PATH, --Path=PATH
              add the given paths to sys.path

SEE ALSO
       The full documentation for cherryd is maintained as a Texinfo manual.  If
       the info and cherryd programs are properly installed at  your  site,  the
       command

              info cherryd

       should give you access to the complete manual.
