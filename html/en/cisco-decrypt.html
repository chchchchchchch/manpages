NAME
       cisco-decrypt - decrypts an obfuscated Cisco vpn client pre-shared key

SYNOPSIS
       cisco-decrypt <obfuscated pre-shared key>

DESCRIPTION
       This  command accompanies vpnc. It decrypts the obfuscated pre-shared key
       from *.pcf-configuration files, which must be specified  on  the  command
       line.

       The result will be printed to STDOUT.
