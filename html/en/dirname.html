NAME
       dirname - strip last component from file name

SYNOPSIS
       dirname NAME
       dirname OPTION

DESCRIPTION
       Output  NAME  with  its  last  non-slash  component  and trailing slashes
       removed; if NAME contains no /'s, output `.' (meaning the current  direc‐
       tory).

       --help display this help and exit

       --version
              output version information and exit

EXAMPLES
       dirname /usr/bin/
              Output "/usr".

       dirname stdio.h
              Output ".".
