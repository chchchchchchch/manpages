NAME
       discover-config — config script

SYNOPSIS
       discover-config [options]

Description
       discover-config reports stuff about the package's configuration.

Options
       --VAR     Print the value of the variable VAR followed by a newline.  See
                 below for a list of available variables.

       -l, --list-vars
                 List the available variables.
