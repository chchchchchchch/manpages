NAME
       disdvi - `disassemble' a (p)TeX or XeTeX DVI file

SYNOPSIS
       disdvi [-p] [dvi_file[.dvi]]
       disdvi -x [xdv_file[.xdv]]
       disdvi -h

DESCRIPTION
       disdvi  interprets  the contents of dvi_file or xdv_file printing the DVI
       commands and text in a human-readable ASCII format on the  standard  out‐
       put.

SEE ALSO
       dvi2tty(1)
