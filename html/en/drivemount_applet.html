NAME
       drivemount_applet - Drive Mount Applet for the GNOME panel.

SYNOPSIS
       drivemount_applet [ --activate-goad-server [=GOAD_ID] ]


DESCRIPTION
       drivemount_applet allows you to quickly and easily mount and unmount var‐
       ious types of drives and file systems on your computer.

       To add this applet to a  Panel,  right-click  on  the  Panel  and  choose
       Applets->Utility->Drive Mount.

OPTIONS
       --activate-goad-server
              (Internal use only) GOAD server ID to activate.

              GOAD_ID is usually the applet name [drivemount_applet]
