SYNOPSIS

       Debug tool for S-expressions

       --decimal
              Print offsets using decimal notation

       --assume-hex
              Assume input is a hex dump

       --verbose
              Show what we are doing

       --version
              Print version of the program and exit

       --help Display this help and exit
