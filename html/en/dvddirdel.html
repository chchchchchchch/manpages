NAME
       dvddirdel  —     Deletes a previously authored DVD directory structure in
       DIR


SYNOPSIS
       dvddirdel [-o DIR]

DESCRIPTION
       dvddirdel deletes a previously authored DVD directory structure in DIR


USAGE
       dvddirdel -o DIR


CAVEATS
       None.
