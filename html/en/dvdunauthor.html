NAME
       dvdunauthor —  Removes DVD-Video file structure


SYNOPSIS
       dvdunauthor [path]

DESCRIPTION
       dvdunauthor removes the DVD-Video file structure


USAGE
       dvdunauthor path


CAVEATS
       None.


BUGS
       None.
