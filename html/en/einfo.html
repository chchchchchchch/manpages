NAME
       einfo - shows information of an EPub document

SYNOPSIS
       einfo [options] epub-document

DESCRIPTION
       einfo is a command line tool to show information of an EPub document.

OPTIONS
       A summary of the available options is included below.

       -d     Enables the debug mode (implies -vvv).

       -h     Shows summary of options.

       -p     Linear print book (normal reading).

       -pp    Prints the whole book.

       -t  tour-id
              Prints the tour tour-id.

       -v     Enables the error messages.

       -vv    Like -v, plus warning messages are enabled.

       -vvv   Like -vv, plus info messages are enabled.
