NAME
       enchant-lsmod  -  Show information about available spell-checking modules
       and dictionaries.

SYNOPSIS
       enchant-lsmod [-lang [language_tag]] [-list-dicts] [-h] [-v]

OPTIONS
       -lang [language_tag]
                 Show dictionary to be used for  given  language_tag.   If  lan‐
                 guage_tag is not given show default dictionary for current lan‐
                 guage setting.

       -list-dicts
                 List preferred dictionaries for language tags.

       -h        Show help info and exit.

       -v        Show version and exit.
