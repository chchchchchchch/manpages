NAME
       extlinux-install - install EXTLINUX on your drive


SYNOPSIS
       extlinux-install DEVICE [EXTLINUX_OPTIONS]


DESCRIPTION
       Install EXTLINUX on your drive.


OPTIONS
       extlinux-install has no options.


HOMEPAGE
       More  information about EXTLINUX and the syslinux project can be found at
       <http://syslinux.zytor.com/>.
