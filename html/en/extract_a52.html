NAME
       extract_a52 - extract ATSC A/52 audio from a MPEG stream.

SYNOPSIS
       extract_a52 [-h] [-s [track]] [-t pid] [file]

DESCRIPTION
       `extract_a52' extracts ATSC A/52 audio from a MPEG stream.  Input is from
       stdin if no file is given.

       -h     display help

       -s track
              set track number (0-7 or 0x80-0x87)

       -t pid use transport stream demultiplexer, pid 0x10-0x1ffe
