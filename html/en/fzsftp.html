NAME
       fzsftp - SFTP connection handler of FileZilla

DESCRIPTION
       fzsftp is part of FileZilla. It is used to handle SFTP (SSH File Transfer
       Protocol) connections. It usually gets called by  FileZilla  and  is  not
       intended to be used directly.

       fzsftp is based on the psftp component of PuTTY.
