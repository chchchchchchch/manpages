NAME
       ggtick - periodic signal-based task scheduler


SYNTAX
       ggtick usecs signal

DESCRIPTION
       ggtick generates the specified signal at usecs-microsecond intervals.

       Warning: the parent process will be kill()ed.

OPTIONS
       usecs  Microseconds to wait between signals.

       signal Integer value of the signal to send.

       help, usage
              Output usage information.

FILES
       /usr/bin/ggtick
