NAME
       gksu-properties - Configure the behaviour of gksu


SYNOPSIS
       gksu-properties



DESCRIPTION
       This manual page documents briefly the gksu-properties command.

       gksu-properties  allows  you  to define how gksu(1) grants the privileges
       and locks your input devices (mouse, keyboard...).


SEE ALSO
       gksu(1), su (1), sudo(1)
