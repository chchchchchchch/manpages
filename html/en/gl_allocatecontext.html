NAME
       gl_allocatecontext - allocate a graphics context


SYNOPSIS
       #include <vgagl.h>

       GraphicsContext *gl_allocatecontext(void);


DESCRIPTION
       Allocate  a  graphics  context. This is preferred to hardcoding a context
       variable in a program since the latter  is  incompatible  with  a  future
       vgagl version that have additional context fields.
