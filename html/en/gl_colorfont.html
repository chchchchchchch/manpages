NAME
       gl_colorfont - change the color of a font


SYNOPSIS
       #include <vgagl.h>

       void gl_colorfont(int fw, int fh, int c, void *fp);


DESCRIPTION
       Set  all  nonzero  pixels  in the expanded font fp with 256 characters of
       size fw and fh each to color c.
