NAME
       gl_compiledboxmasksize - compute the size of a compiled masked box


SYNOPSIS
       #include <vgagl.h>

       int gl_compiledboxmasksize(int w, int h, void *sdp);


DESCRIPTION
       Returns the size of the compiled version of the masked bitmap of size (w,
       h) at sdp that would be generated by compileboxmask(3).

       Compiled bitmaps are only supported in linear 256 color modes.
