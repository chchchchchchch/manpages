NAME
       gl_copybox - copy a rectangular screen area


SYNOPSIS
       #include <vgagl.h>

       void gl_copybox(int x1, int y1, int w, int h , int x2, int y2);


DESCRIPTION
       Copy  the  rectangular  area  at  (x1,  y1)  of  size (w, h), to (x2, y2)
       (screencopy).
