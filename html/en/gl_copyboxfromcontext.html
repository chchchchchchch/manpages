NAME
       gl_copyboxfromcontext - copy rectangular area from another context


SYNOPSIS
       #include <vgagl.h>

       void  gl_copyboxfromcontext(GraphicsContext  *gc,  int x1, int y1, int w,
       int h, int x2, int y2);


DESCRIPTION
       Copy the rectangular area at (x1, y1) in the context gc of size (w, h) to
       position (x2, y2) in the current context.

       This is more efficient than copyboxtocontext(3).
