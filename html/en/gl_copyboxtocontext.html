NAME
       gl_copyboxtocontext - copy a rectangular area to another context


SYNOPSIS
       #include <vgagl.h>

       void  gl_copyboxtocontext(int  x1, int y1, int w, int h , GraphicsContext
       *gc, int x2, int y2);


DESCRIPTION
       Copy the rectangular area at (x1, y1) of size (w, h) to position (x2, y2)
       in the context gc.  If possible use copyboxfromcontext(3).
