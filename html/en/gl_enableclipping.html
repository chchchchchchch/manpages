NAME
       gl_enableclipping - enables clipping


SYNOPSIS
       #include <vgagl.h>

       void gl_enableclipping(void);


DESCRIPTION
       Enable automatic clipping in most functions. No, I don't know which ones.
       However, it works for most.  gl_copybox(3) is report to do no clipping.

       Go figure.
