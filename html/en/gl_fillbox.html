NAME
       gl_fillbox - fill a rectangular area


SYNOPSIS
       #include <vgagl.h>

       void gl_fillbox(int x, int y, int w, int h, int c);


DESCRIPTION
       Fill a rectangular area at position (x, y) with size (w, h) of the screen
       with a single color c.
