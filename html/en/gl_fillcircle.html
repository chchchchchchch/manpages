NAME
       gl_fillcircle - draw a filled circle


SYNOPSIS
       #include <vgagl.h>

       void gl_fillcircle(int x, int y, int r, int c);


DESCRIPTION
       Draw a filled circle of radius r in color c , centered at ( x ,  y ).
