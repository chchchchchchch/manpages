NAME
       gl_freecontext - free a virtual screen


SYNOPSIS
       #include <vgagl.h>

       void gl_freecontext(GraphicsContext *gc);


DESCRIPTION
       Free  the  space allocated for the virtual screen in the given context gc
       which should better be a virtual context.
