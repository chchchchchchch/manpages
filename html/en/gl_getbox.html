NAME
       gl_getbox - copy a rectangular pixmap from the screen to a buffer


SYNOPSIS
       #include <vgagl.h>

       void gl_getbox(int x, int y, int w, int h, void *dp);


DESCRIPTION
       Copy  a  rectangular  bitmap at position (x, y) with size (w, h) from the
       screen to a buffer dp.

       Pixmaps are in row-major order. The destination pixmap has the size w * h
       * BYTESPERPIXEL.
