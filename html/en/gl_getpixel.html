NAME
       gl_getpixel - return the color of a pixel


SYNOPSIS
       #include <vgagl.h>

       int gl_getpixel(int x, int y);


DESCRIPTION
       Returns  the  color of pixel at position (x, y) in the format used in the
       screen memory.

       The color -1 is returned for points outside the clipping window.
