NAME
       gl_getpixelrgb - store color components of a pixel


SYNOPSIS
       #include <vgagl.h>

       void gl_getpixelrgb(int x, int y, int *r, int *g, int *b);


DESCRIPTION
       Store  color  components  from pixel at (x, y) ranging from 0 to 255 into
       integers pointed to by r, g and b.
