NAME
       gl_hline - draw a horizontal line


SYNOPSIS
       #include <vgagl.h>

       void gl_hline(int x1, int y, int x2, int c);


DESCRIPTION
       Draw a horizontal line from point (x1, y) to (x2, y) in color c.  x1 must
       not be larger than x2. If x2<x1 the function does nothing.
