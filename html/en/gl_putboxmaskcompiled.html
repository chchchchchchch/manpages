NAME
       gl_putboxmaskcompiled  -  copy  a compiled masked pixmap to a rectangular
       area


SYNOPSIS
       #include <vgagl.h>

       void gl_putboxmaskcompiled(int x, int y, int w, int h, void *dp);


DESCRIPTION
       Copy the contents of a memory buffer dp to a rectangular bitmap at  posi‐
       tion (x, y) with size (w, h).

       This  function  works like gl_putbox, but does not write pixmap pixels of
       color zero. Also, for faster  operation  the  pixmap  was  compiled  with
       gl_compileboxmask(3) first.
