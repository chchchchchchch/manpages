NAME
       gl_putboxpart - copy a partial pixmap to a rectangular area


SYNOPSIS
       #include <vgagl.h>

       void  gl_putboxpart(int x, int y, int w, int h , int bw, int bh, void *dp
       , int xo, int yo);


DESCRIPTION
       Copy the contents of a memory buffer dp to a rectangular bitmap at  posi‐
       tion (x, y) with size (w, h).

       However,  only a part of size (bw, bh) starting at offset (xo, yo) in the
       pixmap is copied.

       Pixmaps are in row-major order. The source pixmap memory has the size w *
       h * BYTESPERPIXEL.
