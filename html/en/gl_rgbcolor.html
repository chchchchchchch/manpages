NAME
       gl_rgbcolor - return pixel value corresponding to an rgb color


SYNOPSIS
       #include <vgagl.h>

       int gl_rgbcolor(int r, int g, int b);


DESCRIPTION
       Returns pixel value used in the current context that corresponds with the
       given color components. Used by setpixelrgb(3).
