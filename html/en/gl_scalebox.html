NAME
       gl_scalebox - scale a pixmap


SYNOPSIS
       #include <vgagl.h>

       void gl_scalebox(int w1, int h1, void *sdp, int w2, int h2, void *ddp);


DESCRIPTION
       Scale the bitmap of size (w1, h1) at sdp to size (w2, h2) and store it at
       ddp, which must be a large enough buffer.

       The pixel size of the current graphics context is used.
