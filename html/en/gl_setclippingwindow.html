NAME
       gl_setclippingwindow - set the clipping window


SYNOPSIS
       #include <vgagl.h>

       void gl_setclippingwindow(int x1, int y1, int x2, int y2);


DESCRIPTION
       Set  the  clipping  window to the rectangle with top-left corner (x1, y1)
       and bottom-right corner (x2, y2) (inclusively).


SEE ALSO
       svgalib(7), vgagl(7), svgalib.conf(5), threedkit(7),  testgl(1),  gl_dis‐
       ableclipping(3), gl_enableclipping(3).
