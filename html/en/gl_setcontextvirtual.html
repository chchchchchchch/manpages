NAME
       gl_setcontextvirtual - define a virtual context


SYNOPSIS
       #include <vgagl.h>

       void gl_setcontextvirtual(int w, int h, int bpp, int bitspp, void *vbuf);


DESCRIPTION
       Define  the  current graphics context to have a width of w pixels, height
       h, bpp bytes per pixel, bitspp significant color bits per pixel  (8,  15,
       16  or  24),  with the framebuffer at vbuf.  A 4 bytes per pixel context,
       with 24 significant color bits is also valid.
