NAME
       gl_setcontextwidth, gl_setcontextheight - set the dimension of a context


SYNOPSIS
       #include <vgagl.h>

       void gl_setcontextwidth(int w);
       void gl_setcontextheight(int h);


DESCRIPTION
       Set  the size of the current virtual context, effectively clipping every‐
       thing to the top left corner of size (w, h).
