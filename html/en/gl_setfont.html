NAME
       gl_setfont - set the text font to be used


SYNOPSIS
       #include <vgagl.h>

       void gl_setfont(int fw, int fh, void *fp);


DESCRIPTION
       Use  the  font stored as character bitmaps at fp, with characters of size
       (fw, fh), as the basic font for write  operations.  Note  that  the  font
       included in the library must be expanded first, because it is stored bit-
       per-pixel; this is not required if the FONT_COMPRESSED gl_setwritemode(3)
       flag is set.
