NAME
       gl_setfontcolors - set the font colors


SYNOPSIS
       #include <vgagl.h>

       void gl_setfontcolors(int bg, int fg);


DESCRIPTION
       Set  the  background  and foreground colors for the compressed font write
       mode which is set with gl_setwritemode(3).
