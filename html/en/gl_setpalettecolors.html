NAME
       gl_setpalettecolor,  gl_setpalettecolors,  gl_setpalette  - set the color
       palette


SYNOPSIS
       #include <vgagl.h>

       void gl_setpalettecolor(int c, int r, int g, int b);
       void gl_setpalettecolors(int s, int n, void *sp);
       void gl_setpalette(void *sp);


DESCRIPTION
       gl_setpalettecolor sets red, green and blue values (in the range 0 -  63)
       of color c from the color-lookup-table to r, g and b.

       gl_setpalettecolors  sets RGB values of n colors starting at s, which are
       stored as a table of groups of three bytes each at sp.

       gl_setpalette is equivalent to setpalettecolors(0, 256, sp).

       vga_ext_set(3) might change the range of the colors to be set to 0 - 255.
