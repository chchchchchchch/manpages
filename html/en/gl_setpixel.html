NAME
       gl_setpixel, gl_setpixelrgb - draw a pixel


SYNOPSIS
       #include <vgagl.h>

       void gl_setpixel(int x, int y, int c),
       void gl_setpixelrgb(int x, int y, int r, int g, int b);


DESCRIPTION
       gl_setpixel  draws  a  single  pixel  at position (x, y) in color c.  The
       lower 8, 15, 16 or 24 bits of the color are significant, depending on the
       number of colors the current mode supports.

       gl_setpixelrgb  draws a single pixel at position (x, y) with color compo‐
       nents r, g, and b, ranging from 0 to 255. In 256 color mode,  only  mean‐
       ingful if the RGB palette is set (gl_setrgbpalette(3)).
