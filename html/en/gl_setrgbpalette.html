NAME
       gl_setrgbpalette - set a 256-color RGB palette


SYNOPSIS
       #include <vgagl.h>

       void gl_setrgbpalette(void)


DESCRIPTION
       Sets 256-color RGB palette (bits 0 - 2 blue, 3 - 5 green, 6 - 7 red).

       Call this prior to using gl_getpixelrgb(3), gl_rgbcolor(3), or gl_setpix‐
       elrgb(3) in 256 color modes.
