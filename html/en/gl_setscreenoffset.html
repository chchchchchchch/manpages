NAME
       gl_setscreenoffset - set a memory offset for copyscreen


SYNOPSIS
       #include <vgagl.h>

       void gl_setscreenoffset(int o);


DESCRIPTION
       Set the offset in pixels into video memory for copyscreen(3) and copybox‐
       tocontext(3), copyboxfromcontext(3) and thus  allows  for  page-flipping.
       Must  be  a  multiple of the scanline width in bytes. It is reset to zero
       after the completion of copyscreen(3).


SEE ALSO
       svgalib(7), vgagl(7), svgalib.conf(5), threedkit(7), testgl(1),  gl_copy‐
       boxfromcontext(3),        gl_copyboxtocontext(3),       gl_copyscreen(3),
       gl_enablepageflipping(3).
