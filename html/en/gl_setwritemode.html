NAME
       gl_setwritemode - set the font writemode flags


SYNOPSIS
       #include <vgagl.h>

       void gl_setwritemode(int m);


DESCRIPTION
       Sets  the  writemode  flags  for the font routines. It must be the sum of
       exactly one choice out of the following two groups:

       WRITEMODE_MASKED
       WRITEMODE_OVERWRITE (default)
              If WRITEMODE_MASKED is set, only foreground pixels of the font are
              used for write operations; the screen background is not erased.


       FONT_COMPRESSED
       FONT_EXPANDED (default)
              If  FONT_COMPRESSED  is  set,  text writes will use the compressed
              bit-per-pixel font rather than the expanded font (which  would  be
              slightly faster).
