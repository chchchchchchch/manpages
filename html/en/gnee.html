NAME
       gnee - record, replays or distributes X11 data

SYNOPSIS
       gnee


DESCRIPTION
       gnee  is the graphical frontend to GNU Xnee. If you want more information
       about GNU Xnee read the GNU Xnee manual and the cnee info pages. GNU Xnee
       consists of a command lie program (cnee), a graphical frontend (gnee) and
       a Gnome Panel Applet (pnee).
