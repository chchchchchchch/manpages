NAME
       gnome-screensaver-command - controls GNOME screensaver

SYNOPSIS
       gnome-screensaver-command [OPTION...]

DESCRIPTION
       gnome-screensaver-command  is  a  tool for controlling an already running
       instance of gnome-screensaver.

OPTIONS
       --exit Causes the screensaver to exit gracefully

       -q, --query
              Query the state of the screensaver

       -t, --time
              Query the length of time the screensaver has been active

       -l, --lock
              Tells the running screensaver process to lock the  screen  immedi‐
              ately

       -a, --activate
              Turn the screensaver on (blank the screen)

       -d, --deactivate
              If  the  screensaver  is  active  then deactivate it (un-blank the
              screen)

       -V, --version
              Version of this application
