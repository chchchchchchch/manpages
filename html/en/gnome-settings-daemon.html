NAME
       gnome-settings-daemon - Handles the GNOME session settings

SYNOPSIS
       gnome-settings-daemon  [--debug]  [--no-daemon]  [--gconf-prefix] [--dis‐
       play=DISPLAY]

DESCRIPTION
       gnome-settings-daemon is responsible for setting the  various  preference
       parameters of a GNOME session and the applications that run under it.

OPTIONS
       Help options

       -?, --help
              Show this help message

       Application options

       --debug
              Enable debugging code

       --no-daemon
              Do not detach the daemon process from its controlling terminal

       --gconf-prefix
              Specify  a custom GConf settings prefix for settings manager plug‐
              ins

       --display=DISPLAY
              X display to use
