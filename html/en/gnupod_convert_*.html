NAME
       gnupod_convert_* — convert files to your desired audio file format

DESCRIPTION
       The  gnupod_convert_*  applications are used by gnupod_addsond internaly.
       Do not use them directly.

       This manual page was written for  the  Debian  distribution  because  the
       original program does not have a manual page.  Instead, it has documenta‐
       tion in the GNU       Info format; see below.

       gnupod_addsong (1).
