NAME
       gpg-error - program to convert GnuPG error numbers to text

SYNOPSIS
       gpg-error NUMBER...

DESCRIPTION
       gpg-error  is  a  tool that converts numeric GnuPG errors to a short one-
       line textual description.

OPTIONS
       gpg-error does not accept any options.
