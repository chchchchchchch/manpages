NAME
       grub-mklayout - generate a GRUB keyboard layout file

SYNOPSIS
       grub-mklayout [OPTIONS]

DESCRIPTION
       -i, --input
              set input filename. Default is STDIN

       -o, --output
              set output filename. Default is STDOUT

       -h, --help
              display this message and exit.

       -V, --version
              print version information and exit.

       -v, --verbose
              print verbose messages.

REPORTING BUGS
       Report bugs to <bug-grub@gnu.org>.

SEE ALSO
       grub-mkconfig(8)

       The  full documentation for grub-mklayout is maintained as a Texinfo man‐
       ual.  If the info and grub-mklayout programs are  properly  installed  at
       your site, the command

              info grub-mklayout

       should give you access to the complete manual.
