NAME
       grub-mkpasswd-pbkdf2 - generate hashed password for GRUB

SYNOPSIS
       grub-mkpasswd-pbkdf2 [OPTIONS]

OPTIONS
       -c number, --iteration-count=number
              Number of PBKDF2 iterations

       -l number, --buflen=number
              Length of generated hash

       -s number, --salt=number
              Length of salt

REPORTING BUGS
       Report bugs to <bug-grub@gnu.org>.

SEE ALSO
       grub-mkconfig(8)

       The  full  documentation for grub-mkpasswd-pbkdf2 is maintained as a Tex‐
       info manual.  If the info and grub-mkpasswd-pbkdf2 programs are  properly
       installed at your site, the command

              info grub-mkpasswd-pbkdf2

       should give you access to the complete manual.
