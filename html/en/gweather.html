NAME
       gweather - Weather Applet for the GNOME panel.

SYNOPSIS
       gweather [ --activate-goad-server [=GOAD_ID] ]


DESCRIPTION
       gweather  displays  the  current  temperature  and  weather conditions in
       numeric and iconified form inside the applet.

       To add this applet to a  Panel,  right-click  on  the  Panel  and  choose
       Applets->Utility->GNOME Weather

OPTIONS
       --activate-goad-server
              (Internal use only) GOAD server ID to activate.

              GOAD_ID is usually the applet name [gweather]
