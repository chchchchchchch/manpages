NAME
       htdigconfig - script to create fuzzy databases for ht://Dig

SYNOPSIS
       htdigconfig

DESCRIPTION
       htdigconfig  is  a  script  to  create fuzzy databases such as word2root,
       root2word and synonyms databases for the ht://Dig search engine.

SEE ALSO
       Please  refer  to   the   HTML   pages   (in   the   htdig-doc   package)
       /usr/share/doc/htdig-doc/html/index.html  for  a  detailed description of
       ht://Dig and its commands.
