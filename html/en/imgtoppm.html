NAME
       imgtoppm - convert an Img-whatnot file into a portable pixmap

SYNOPSIS
       imgtoppm [imgfile]

DESCRIPTION
       Reads  an  Img-whatnot file as input.  Produces a portable pixmap as out‐
       put.  The Img-whatnot toolkit is available  for  FTP  on  venera.isi.edu,
       along with numerous images in this format.

SEE ALSO
       ppm(5)

AUTHOR
       Based on a simple conversion program posted to comp.graphics by Ed Falk.
