NAME
       input-linux-evdev - Linux event interface input

SYNOPSIS
       input-linux-evdev: [<device>]


DESCRIPTION
       Reads  events from the Linux event interface (/dev/input/eventX) and con‐
       verts them to LibGII events.

OPTIONS
       device Specifies the device to use, the default is /dev/input/event0.
