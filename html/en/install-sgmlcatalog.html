NAME
       install-sgmlcatalog - maintain transitional SGML catalog

SYNOPSIS
       install-sgmlcatalog --remove package

DESCRIPTION
       install-sgmlcatalog  removes  entries  from the transitional SGML catalog
       file /etc/sgml/transitional.cat.

OPTIONS
       --remove package
              Indicates to remove the entry for the package package.
