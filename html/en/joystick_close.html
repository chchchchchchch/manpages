NAME
       joystick_close - close a joystick

SYNOPSIS
       #include <vgajoystick.h>

       void joystick_close(int joydev);


DESCRIPTION
       closes the joystick joydev which was opened with joystick_init(3).

       joydev  =  -1  closes  all currently open joystick devices. Note that any
       calibration data is lost if you close a joystick.


CAVEATS
       This function is only available in ELF versions of svgalib. Due to  back‐
       wards compatibility issues it cannot be used with shared a.out libs.
