NAME
       list-unreleased - display UNRELEASED packages

SYNOPSIS
       list-unreleased [arg ...] [path ...]

DESCRIPTION
       Searches for packages whose changelogs indicate there are pending changes
       (UNRELEASED) and either lists them or  displays  the  relevant  changelog
       entry.

       By  default  it  searches  for packages under the current directory. If a
       path is specified it will look for packages under that directory instead.

OPTIONS
       -c     Display pending changes.

       -R     Don't recurse into subdirectories looking for packages.

SEE ALSO
       debchange(1)
