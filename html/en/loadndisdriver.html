NAME
       loadndisdriver  -  Userspace ndis driver loader for the ndiswrapper Linux
       kernel module

SYNOPSIS
       loadndisdriver

DESCRIPTION
       loadndisdriver is a support program for the ndiswrapper Linux kernel mod‐
       ule. The ndiswrapper kernel module uses it to load ndis drivers that have
       been installed via the ndiswrapper(8) tool.

       loadndisdriver should not be used directly, only the kernel module should
       use  it. Therefore, it's options are volatile, and subject to change dur‐
       ing the development of the ndiswrapper kernel module.

       The /etc/ndiswrapper directory is expected to contain the ndis files that
       loadndisdriver requires.

SEE ALSO
       ndiswrapper(8)
