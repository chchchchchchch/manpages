NAME
       mcheck - verify all files on an MS-DOS formatted disk

SYNOPSIS
       mcheck [msdosdrive]

DESCRIPTION
       mcheck  is a script that verifies all files on a MS-DOS formatted disk by
       reading them using mtype(1)

       The optional argument specifies the MS-DOS drive letter of the disk to be
       checked. A: is used by default.
