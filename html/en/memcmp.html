NAME
       memcmp - compare memory areas

SYNOPSIS
       #include <string.h>

       int memcmp(const void *s1, const void *s2, size_t n);

DESCRIPTION
       The  memcmp()  function  compares  the first n bytes (each interpreted as
       unsigned char) of the memory areas s1 and s2.  It returns an integer less
       than,  equal to, or greater than zero if s1 is found, respectively, to be
       less than, to match, or be greater than s2.

RETURN VALUE
       The memcmp() function returns an integer less than, equal to, or  greater
       than  zero  if the first n bytes of s1 is found, respectively, to be less
       than, to match, or be greater than the first n bytes of s2.

CONFORMING TO
       SVr4, 4.3BSD, C89, C99, POSIX.1-2001.
