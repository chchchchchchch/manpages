NAME
       memdiskfind - utility to search for a MEMDISK instance


SYNOPSIS
       memdiskfind


DESCRIPTION
       memdiskfind is a simple utility to search for a MEMDISK instance and out‐
       put the parameters needed to use the "phram" driver in Linux to map it.


SEE ALSO
       syslinux(1)
