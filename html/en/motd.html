NAME
       motd - message of the day

DESCRIPTION
       The contents of /etc/motd are displayed by pam_motd(8) after a successful
       login but just before it executes the login shell.

       The abbreviation "motd" stands for "message of the day",  and  this  file
       has  been traditionally used for exactly that (it requires much less disk
       space than mail to all users).

       On Debian GNU/Linux, the content of /run/motd.dynamic is also  displayed.
       This file is generated by /etc/init.d/motd at boot.

FILES
       /etc/motd
       /run/motd.dynamic
       /etc/init.d/motd

SEE ALSO
       login(1), issue(5) pam_motd(8)
