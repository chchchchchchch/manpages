NAME
       mouse_seteventhandler,  mouse_setdefaulteventhandler  - set a mouse event
       handler

SYNOPSIS
       #include <vgamouse.h>

       typedef void (*__mouse_handler) (int button, int dx, int dy, int dz , int
       drx,  int  dry, int drz); /* is already in #include <vgamouse.h> included
       */

       void mouse_seteventhandler(__mouse_handler handler);
       void mouse_setdefaulteventhandler(void);


DESCRIPTION
       mouse_update(3) and mouse_waitforupdate(3) will call  this  function  for
       each change of mouse state and you can do your own book keeping and maybe
       even draw a mouse cursor.

       mouse_setdefaulteventhandler() activates the default handler.
