NAME
       mouse_setscale - sets a mouse scale factor

SYNOPSIS
       #include <vgamouse.h>

       void mouse_setscale(int s);


DESCRIPTION
       This  routine  sets  the  scale factor between the motion reported by the
       mouse and the size of one pixel.  The larger the scale is, the slower the
       mouse cursor appears to move.  The scale may be set to any non-zero inte‐
       ger.  Negative scales result in flipped axes.   Currently,  there  is  no
       support  for  scale  factors  between  0 and 1 which would make the mouse
       appear faster than usual.

       If this routine is never called, scale s defaults to 1.
