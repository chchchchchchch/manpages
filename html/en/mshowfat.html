Name
       mshowfat - shows FAT clusters allocated to file



Note of warning
       This manpage has been automatically generated from mtools's texinfo docu‐
       mentation, and may not be entirely accurate or complete.  See the end  of
       this man page for details.

Description
       The mshowfat command is used to display the FAT entries for a file.  Syn‐
       tax:

       mshowfat [-o offset] files


       If no offset is given, a list of all clusters occupied  by  the  file  is
       printed. If an offset is given, only the number of the cluster containing
       that offset is printed.
