NAME
       mtrace - Interpret output from MALLOC_TRACE

SYNOPSIS
       mtrace [OPTION]... [Binary] MtraceData

DESCRIPTION
       The mtrace interprets the output from when the MALLOC_TRACE environment
       variable is set.

AUTHOR
       mtrace was written by Ulrich Drepper as part of the GNU C Library.
