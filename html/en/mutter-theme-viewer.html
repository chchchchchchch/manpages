NAME
       mutter-theme-viewer - view mutter themes

SYNOPSIS
       mutter-theme-viewer [ THEMENAME ]

DESCRIPTION
       mutter-theme-viewer allows you to preview any installed Mutter theme.

       When  designing  a  new  Mutter theme, you can use mutter-theme-viewer to
       measure the performance of a window frame  option,  and  to  preview  the
       option.

OPTIONS
       THEMENAME
              Name  of  the theme to be shown (Atlanta by default).  It is case-
              sensitive.

FILES
       /usr/share/themes
              system themes directory

       /usr/share/themes/*/mutter-1/mutter-theme-1.xml
              theme specification file
