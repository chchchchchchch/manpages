NAME
       mutter-window-demo - demo of window features

SYNOPSIS
       mutter-window-demo

DESCRIPTION
       This  program  demonstrates various kinds of windows that window managers
       and window manager themes should handle.

       Be sure to tear off the menu and toolbar, those are also a  special  kind
       of window.

AUTHOR
       This  manual page was written by Jose M. Moya <josem@die.upm.es>, for the
       Debian  GNU/Linux system (but may be used by others).
