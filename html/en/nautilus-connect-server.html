NAME
       nautilus-connect-server - To Access a remote server

SYNOPSIS
       nautilus-connect-server [options] [URI]

DESCRIPTION
       This manual page documents briefly the nautilus-connect-server command.

       Nautilus Connect Server is the connection manager for the GNOME desktop.

       You  can  use  the  file  manager to access a remote server, be it an FTP
       site, a Windows share, a WebDav server or an SSH server.

OPTIONS
       Nautilus follows the usual GNU command line  syntax,  with  long  options
       starting with two dashes (`-'). A summary of options is included below.

       --help Show a summary of options.

       --version
              Show version.

       Other standard GNOME options not listed here are also supported.

SEE ALSO
       Nautilus  Connect Server documentation can be found from the "Help" menu,
       or  by  pressing  the  F1  key.  Nautilus   also   has   a   website   at
       http://www.gnome.org/projects/nautilus/
