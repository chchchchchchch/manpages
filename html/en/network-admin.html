NAME
       network-admin - Network Administration Tool

SYNOPSIS
       network-admin [OPTION...]

DESCRIPTION
       network-admin is part of the GNOME system tools, a set of tools to easily
       access and manage system configuration.

       network-admin allows you to specify the way your system connects to other
       computers and to the Internet.

OPTIONS
       --display=DISPLAY
              X display to use

       network-admin also accepts the standard GNOME and GTK options.
