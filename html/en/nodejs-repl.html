NAME
       nodejs-repl - Nodejs read-eval-print-loop

SYNOPSIS
       echo [SHORT-OPTION]... [STRING]...

DESCRIPTION
       nodejs-repl provides a read eval print loop for nodejs.

       To provide a better interface, nodejs-repl uses rlwrap.

SEE ALSO
       nodejs(1), rlwrap(1)
