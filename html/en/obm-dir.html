NAME
       obm-dir - a pipe-menu for openbox

SYNTAX
       obm-dir <directory> <command>

DESCRIPTION
       Obm-dir  is  a pipe menu for openbox.  It builds a menu from a directory,
       alphabetically sorted and can be used inside your openbox's menu to  gen‐
       erate menus dynamically.

       More     information     about     pipe-menus    can    be    found    on
       http://icculus.org/openbox/index.php/Openbox:Pipemenus

EXAMPLES
       obm-dir /home/bull/wallpaper fbsetbg
