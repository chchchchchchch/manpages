NAME
       obm-nav - a pipe-menu for openbox

SYNTAX
       obm-nav <root> <filer> <terminal>

DESCRIPTION
       Obm-moz is a pipe menu for openbox.  It recursively builds a menu to nav‐
       igate directories.

       More    information    about    pipe-menus    can     be     found     on
       http://icculus.org/openbox/index.php/Openbox:Pipemenus

EXAMPLES
       obm-nav /home/bull rox xterm
