NAME
       obm-xdg - a pipe-menu for openbox

SYNTAX
       obm-xdg [options] /path/to/system.menu

DESCRIPTION
       Obm-xdg is a free desktop menus implementation for openbox.

       More     information     about     pipe-menus    can    be    found    on
       http://icculus.org/openbox/index.php/Openbox:Pipemenus

OPTIONS
       --import  Creates a file called "gnome-menus.xml"

       --replace Replaces the Openbox menu with the Gnome menus
