NAME
       openbox-kde-session  —  Run a KDE session with Openbox as the window man‐
       ager


DESCRIPTION
       openbox-kde-session runs a KDE session with Openbox as  the  window  man‐
       ager.

       openbox-kde-session does not take any command line arguments.

SEE ALSO
       openbox(1), openbox-gnome-session(1), openbox-session(1).


       The   program's   full   documentation   is  available  on  the  website:
       http://openbox.org/

       Please report bugs to: http://bugzilla.icculus.org/
