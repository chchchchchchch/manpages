NAME
       panel-test-applets —  display installed applets

SYNOPSIS
       panel-test-applets [--help ]

DESCRIPTION
       panel-test-applets  display  installed  applets  in a dialog box and does
       nothing else.

OPTIONS
       --help    Display help

SEE ALSO
       gnome-options(7)
