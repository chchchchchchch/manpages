NAME
       pbmtomgr - convert a portable bitmap into a MGR bitmap

SYNOPSIS
       pbmtomgr [pbmfile]

DESCRIPTION
       Reads a portable bitmap as input.  Produces a MGR bitmap as output.

SEE ALSO
       mgrtopbm(1), pbm(5)
