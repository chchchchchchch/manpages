NAME
       pdb3.2 - the Python debugger

SYNOPSIS
       pdb3.2 script [...]

DESCRIPTION
       See  /usr/lib/python3.2/pdb.doc  for  more information on the use of pdb.
       When the debugger is started, help is available via the help command.

SEE ALSO
       python3.2(1). Chapter 9 of  the  Python  Library  Reference  (The  Python
       Debugger).     Available     in     the    python3.2-doc    package    at
       /usr/share/doc/python3.2/html/lib/module-pdb.html.
