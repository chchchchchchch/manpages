NAME
       pnm - portable anymap file format

DESCRIPTION
       The pnm programs operate on portable bitmaps, graymaps, and pixmaps, pro‐
       duced by the pbm, pgm, and ppm segments.  There is no file format associ‐
       ated with pnm itself.
