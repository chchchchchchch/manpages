NAME
       pnmtile - replicate a portable anymap into a specified size

SYNOPSIS
       pnmtile width height [pnmfile]

DESCRIPTION
       Reads  a  portable anymap as input.  Replicates it until it is the speci‐
       fied size, and produces a portable anymap as output.

SEE ALSO
       pnm(5)
