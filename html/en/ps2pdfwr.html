NAME
       ps2pdfwr  -  Convert  PostScript  to  PDF  without specifying Compatibil‐
       ityLevel, using ghostscript

SYNOPSIS
       ps2pdfwr  [options...] {input.[e]ps|-} [output.pdf|-]

DESCRIPTION
       This wrapper script invokes gs(1) with following arguments

                        -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite

       as well as the appropriate -dOutputFile argument, all preceded  and  fol‐
       lowed by any command-line arguments. Finally, the security option -dSAFER
       is prepended before all the other options.

       The version-specific ps2pdf scripts all invoke this one with the addition
       of the respective compatibility level option.

SEE ALSO
       gs(1), ps2pdf(1)
