NAME
       qrttoppm - convert output from the QRT ray tracer into a portable pixmap

SYNOPSIS
       qrttoppm [qrtfile]

DESCRIPTION
       Reads a QRT file as input.  Produces a portable pixmap as output.

SEE ALSO
       ppm(5)
