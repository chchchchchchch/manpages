NAME
           select-default-wordlist - select default wordlist

SYNOPSIS
        select-default-wordlist

DESCRIPTION
       This program will make debconf always ask the shared question about the
       default wordlist to be used in your system according to the installed
       ones, and will do the appropriate settings.

       Calls internally update-default-wordlist.

       The dictionaries-common policy document
