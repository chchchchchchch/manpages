NAME
       services-admin - Services Administration Tool

SYNOPSIS
       services-admin [OPTION...]

DESCRIPTION
       services-admin  is part of the GNOME system tools, a set of tools to eas‐
       ily access and manage system configuration.

       services-admin allows you to specify which services will be started  dur‐
       ing the system boot process.

OPTIONS
       services-admin accepts the standard GNOME and GTK options.
