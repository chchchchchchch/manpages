NAME
       soundex - Soundex

SYNOPSIS
       package require Tcl  8.2

       package require soundex  ?1.0?

       ::soundex::knuth string

_________________________________________________________________

DESCRIPTION
       This  package  provides  soundex algorithms which allow the comparison of
       words based on their phonetic likeness.

       Currently only an algorithm by Knuth is provided, which is tuned to  eng‐
       lish names and words.

       ::soundex::knuth string
              Computes  the soundex code of the input string using Knuth's algo‐
              rithm and returns it as the result of the command.

EXAMPLES
                  % ::soundex::knuth Knuth
                  K530
