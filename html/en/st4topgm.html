NAME
       st4topgm - convert an SBIG ST4 format file into a portable graymap

SYNOPSIS
       st4topgm [st4file]

DESCRIPTION
       Reads  an an image file in the native format used by the SBIG ST-4 astro‐
       nomical CCD camera, and produces a portable  graymap  as  output.   Addi‐
       tional  information  on SBIG cameras and documentation of the file format
       is available at the Web site: http://www.sbig.com/

SEE ALSO
       pgm(5) sbigtopgm(1)
