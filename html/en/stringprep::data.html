NAME
       stringprep::data - stringprep data tables, generated, internal

SYNOPSIS
       package require Tcl  8.3

       package require stringprep::data  1.0.1

_________________________________________________________________

DESCRIPTION
       The  stringprep::data  package  is  a helper for stringprep, providing it
       with the data tables needed to perform its functions. It is  an  internal
       package  which  should not be accessed on its own. Because of that it has
       no publicly documented API either. Its implementation is generated  by  a
       script.
