NAME
       tinyirc - A tiny IRC client

SYNOPSIS
       tinyirc [-dumb] [nick] [server] [port]

DESCRIPTION
       tinyirc  is  a stripped down IRC client, providing basic IRC commands and
       an curses-based interface.

OPTIONS
       -dumb  Do not use the curses interface; simply write to stdout  and  read
              commands from stdin.

ENVIRONMENT
       IRCSERVER
              Sets the default server to connect to.

       IRCNICK
              Sets the default nickname to use.

SEE ALSO
       RFC1459
