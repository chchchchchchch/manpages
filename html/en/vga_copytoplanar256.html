NAME
       vga_copytoplanar256 - copy linear pixmap into Mode X video memory

SYNOPSIS
       #include <vga.h>

       void  vga_copytoplanar256(unsigned char *virtual, int pitch, int voffset,
       int vpitch, int w, int h);


DESCRIPTION
       Copy part of a linear virtual screen in system memory at *virtual to Mode
       X  style  video memory.  pitch is the logical width of the virtual screen
       (in pixels (which are one byte large)), voffset  is  the  address  offset
       into  video  memory,  vpitch is the logical scanline width of the screen.
       (w, h) is the width and height of the area to copy in pixels (w must be a
       multiple of 4).
