NAME
       vga_disabledriverreport - makes svgalib not emit any startup messages

SYNOPSIS
       #include <vga.h>

       void vga_disabledriverreport(void);


DESCRIPTION
       Usually svgalib prints the name of the hardware detected or forced to the
       screen during startup. This and other  informational  messages  are  sup‐
       pressed when vga_disabledriverreport() was called.
