NAME
       vga_drawscanline - draw a horizontal line of pixels

SYNOPSIS
       #include <vga.h>

       int vga_drawscanline(int line , unsigned char *color);


DESCRIPTION
       draws  a horizontal line over the whole screen in the line with y coordi‐
       nate line.

       The colors drawn are taken from color[0] .. color[length - 1].

       This is a simpler frontend to vga_drawscansegment(3).
