NAME
       vga_dumpregs - dump the contents of the SVGA registers

SYNOPSIS
       #include <vga.h>

       int vga_dumpregs(void);


DESCRIPTION
       This  function  dumps  the  current values of all SVGA registers in a way
       which can by copied in a source file.  This  is  for  debugging  purposes
       only.

SEE ALSO
       svgalib(7),  vgagl(7),  libvga.config(5),  vga_init(3),  vga_dumpregs(3),
       dumpreg(1)
