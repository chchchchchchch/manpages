NAME
       vga_getch - wait for a key

SYNOPSIS
       #include <vga.h>

       int vga_getch(void);


DESCRIPTION
       waits for a key press just like getchar(3) would.

       For  a  non  blocking  check  for  a  keypress  use  vga_getkey(3) or use
       vga_waitevent(3) if you want to wait for several events to occur at once.
