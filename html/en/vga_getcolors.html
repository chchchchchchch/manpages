NAME
       vga_getxdim, vga_getydim, vga_getcolors - return the current screen reso‐
       lution

SYNOPSIS
       #include <vga.h>

       int vga_getxdim(void);
       int vga_getydim(void);
       int vga_getcolors(void);


DESCRIPTION
       These function return the currently selected screen resolution resp. num‐
       ber of displayable colors.

       svgalib(7),  vgagl(7),  libvga.config(5),  vga_setmode(3), vga_getcurren‐
       tode(3), vga_getmodeinfo(3), vga_getgraphmem(3)
