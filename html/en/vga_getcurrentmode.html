NAME
       vga_getcurrentmode - returns the current video mode

SYNOPSIS
       #include <vga.h>

       int vga_getcurrentmode(void);


DESCRIPTION
       Returns  the current video mode.  See vga_setmode(3) for a list of prede‐
       fined return values and there meaning.
