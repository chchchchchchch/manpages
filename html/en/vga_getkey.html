NAME
       vga_getkey - read a character from the keyboard without waiting

SYNOPSIS
       #include <vga.h>

       int vga_getkey(void);


DESCRIPTION
       Read  a  character  from the keyboard without waiting; returns zero if no
       key pressed and the ASCII value otherwise. This function must not be used
       in  raw keyboard mode. In this situation use keyboard_getstate(3) or key‐
       board_keypressed(3) instead.
