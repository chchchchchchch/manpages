NAME
       vga_getpixel - get a pixels value from the screen

SYNOPSIS
       #include <vga.h>

       int vga_getpixel(int x, int y);


DESCRIPTION
       Read  a  pixels value from the screen. This function is very inefficient,
       you  should  consider  doing  direct  screen  access   instead   or   use
       vga_getscansegment(3) to handle more than one pixel.
