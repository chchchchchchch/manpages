NAME
       vga_hasmode - returns if a video mode is supported

SYNOPSIS
       #include <vga.h>

       int vga_hasmode(int mode);


DESCRIPTION
       The  function returns a non-zero value if the given mode is supported and
       0 else.  See vga_setmode(3) for a list  of  predefined  mode  values  and
       their meaning.  vga_modeinfo(3) can be used to find out details about the
       specific mode.
