NAME
       vga_lastmodenumber - returns the last video mode number

SYNOPSIS
       #include <vga.h>

       int vga_lastmodenumber(void);


DESCRIPTION
       Returns the last video mode number. As video mode numbers start with 1 (0
       is for textmode) this allows an application to scan through all  possible
       modes portable with vga_modeinfo(3).

       For example the vgatest(6) demo does just that.

       #include <vga.h> does include a #define GLASTMODE vga_lastmodenumber().
