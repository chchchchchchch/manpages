NAME
       vga_lockvc - disables virtual console switching for safety

SYNOPSIS
       #include <vga.h>

       void vga_lockvc(void);


DESCRIPTION
       Disables  virtual  console  switching  for safety. Recommended when doing
       register I/O or low-level device handling. Used by mouse functions.  Also
       useful in conjunction with background running.
