NAME
       vga_runinbackground_version  -  returns  the version of the current back‐
       ground support

SYNOPSIS
       #include <vga.h>

       int vga_runinbackground_version(void);


DESCRIPTION
       returns the version of the current background support which  is  1  (uses
       proc-fs).  In  case  svgalib is compiled without background support, 0 is
       returned. See vga_runinbackground(3) for more information  on  background
       support.


SEE ALSO
       svgalib(7),   vgagl(7),   libvga.config(5),   bg_test(6),  vga_runinback‐
       ground(3)
