NAME
       vga_setrgbcolor - set the current color

SYNOPSIS
       #include <vga.h>

       void vga_setrgbcolor(int r, int g, int b);


DESCRIPTION
       Set the current color to RGB values r, g, and b, each in the range 0-255.
       Does not make sense in 256  or  less  color  modes.  Use  vga_setcolor(3)
       there. The color is used by vga_drawpixel(3) and vga_drawline(3).

       The fun(6) and vgatest(6) demos show the basic use of this function.
