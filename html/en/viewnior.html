NAME
       viewnior - simple, fast and elegant image viewer

SYNOPSIS
       viewnior [FILES|FOLDERS]...

DESCRIPTION
       Viewnior  is an image viewer program. Created to be simple, fast and ele‐
       gant. It's minimalistic interface  provides  more  screenspace  for  your
       images.

OPTIONS
       -?, --help
              Show this help and exit

       --version
              Show version information and exit
