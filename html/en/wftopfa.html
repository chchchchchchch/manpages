NAME
       wftopfa - Convert a Wadalab base font to Postscript .PFA (or .PFB) format
       using ghostscript

SYNOPSIS
       wftopfa fontname

DESCRIPTION
       This script invokes gs(1) to convert a Wadalab base  font  to  Postscript
       .PFA (or .PFB) format.

SEE ALSO
       gs(1)
