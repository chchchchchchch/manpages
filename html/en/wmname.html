NAME
       wmname - prints/sets window manager name


SYNOPSIS
       wmname [WM_NAME]


DESCRIPTION
       wmname  prints/sets  the  window manager name property of the root window
       similiar to how hostname(1) behaves.  wmname is a  nice  utility  to  fix
       problems with JDK versions and other broken programs assuming a reparent‐
       ing window manager for instance.


OPTIONS
       wmname has no options.
