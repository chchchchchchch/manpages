NAME
       read, write, etc - basic file operations

METHODS
   result = $handle->close
   list = $handle->get_file_info ($options)
       ·   $options (Gnome2::VFS::FileInfoOptions)

       Returns a GnomeVFSResult and a GnomeVFSFileInfo.

   result = $handle->forget_cache ($offset, $size)
       ·   $offset (unsigned)

       ·   $size (unsigned)

       Since: vfs 2.12

   list = $handle->read ($bytes)
       ·   $bytes (unsigned)

       Returns a GnomeVFSResult, the number of bytes read and the buffer
       containing the read content.

   result = $handle->seek ($whence, $offset)
       ·   $whence (Gnome2::VFS::SeekPosition)

       ·   $offset (unsigned)

   list = $handle->tell
       Returns a GnomeVFSResult and the offset.

   result = $handle->truncate ($length)
       ·   $length (unsigned)

   list = $handle->write ($buffer, $bytes)
       ·   $buffer (string)

       ·   $bytes (unsigned)

       Returns a GnomeVFSResult and the number of bytes written.
