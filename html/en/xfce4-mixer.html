NAME
       xfce4-mixer - a volume control application for Xfce

SYNOPSIS
       xfce4-mixer

DESCRIPTION
       xfce4-mixer is a volume control application for the Xfce Desktop Environ‐
       ment. This software package contains both a volume control plugin for the
       Xfce  Panel and a standalone mixer application. You should be able to run
       the latter from the menu system.
